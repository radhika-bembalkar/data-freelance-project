# Web Scraper for Pricing Data

## Overview
Automated Python script to scrape product/market pricing data from e-commerce websites. Built during freelance consulting (2019-2021) for SME clients needing competitive benchmarking.

## Features
- HTTP requests with error handling
- HTML parsing using BeautifulSoup
- Data cleaning (deduplication, standardization)
- CSV export for analysis
- Logging and progress tracking
